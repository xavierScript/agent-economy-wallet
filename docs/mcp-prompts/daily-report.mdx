title: Overview
description: "Model Context Protocol Prompts for Agentic Wallets"
---

Provides the LLM with perfectly formatted systemic instructions and predefined task contexts.

- `daily-report`: Instructs the agent on how to summarize daily vault activities and profits.
- `risk-assessment`: Evaluates whether a proposed on-chain transaction is safe based on the policy engine rules.
- `security-audit`: In-depth prompt for reviewing the smart contract and authority statuses of an SPL token.

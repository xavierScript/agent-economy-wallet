MCP Resources act as read-only documents and data streams the AI agent can subscribe to or read dynamically.

## Supported Resources

### `wallet://audit-logs`

Provides the most recent 50 audit log entries across all wallets. Includes timestamps, actions, success/failure status, and transaction signatures.

### `wallet://audit-logs/{wallet_id}`

Dynamically generated resource to retrieve logs for a specific configured wallet. Agents read this to understand their historical performance, confirm whether recent transactions cleared the blockchain safely, or view what actions the Policy Engine has rejected.

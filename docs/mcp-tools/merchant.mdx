The MCP Tools provide AI agents with a standardized interface to interact directly with the wallet core. Agents connected to the MCP Server can parse the tool descriptions and schemas to execute complex financial behaviors.

## Available Tool Categories

- **Merchant Tools**: Tools for operating as a paid service provider (validating x402 receipts, issuing challenges).
- **Payment Tools**: Tools for fulfilling x402 challenges and sending micro-transactions.
- **Token Tools**: Spl token interactions, mint data lookups.
- **Transfer Tools**: Sending SOL or tokens.
- **Wallet Tools**: Key management details, balance checks, simulated transactions.

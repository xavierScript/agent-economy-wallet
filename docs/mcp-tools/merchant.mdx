The MCP Tools provide AI agents with a standardized interface to interact directly with the wallet core. Agents connected to the MCP Server can parse the tool descriptions and schemas to execute complex financial behaviors.

## Supported Merchant Tools

The Merchant tools simulate premium data feeds that another agent might access behind an x402 paywall. When a client agent invokes these tools without providing a `receipt_signature`, the tool will reject the execution and return the required cost, signaling to the client agent that it must first invoke its payment tools to settle the charge on-chain.

### `analyze_token_security`

**PREMIUM:** Analyzes an SPL token's mint account to check for active `mintAuthority` and `freezeAuthority`.

- **Input Parameters:** `mintAddress`, `receipt_signature` (optional)
- **Cost:** 0.1 USDC

### `fetch_token_price`

**PREMIUM:** Provides real-time pricing data for SPL tokens by querying the CoinGecko aggregator natively.

- **Input Parameters:** `mintAddress`, `vsCurrency` (default usdc), `receipt_signature` (optional)
- **Cost:** 0.05 USDC

Welcome to Agent Economy Wallet! This project provides a robust, hybrid **Agent-to-Agent Economy Server** built for the Solana hackathon. It empowers AI agents to seamlessly orchestrate financial processes, execute low-latency payments (via x402), and natively interact using Model Context Protocol (MCP) tools.

## What is Agent Economy Wallet?

Agent Economy Wallet bridges the gap between AI orchestration and on-chain settlements. It allows agents to act autonomously with cryptographic guarantees and verifiable safety. By bringing together the Model Context Protocol (MCP) and Solana's fast execution, it creates a fertile ground for agentic economies.

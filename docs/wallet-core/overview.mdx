---
title: Overview
description: "Core Wallet Engine Components"
---

The `wallet-core` package encapsulates all Solana-specific logic, cryptography, guardrails, and protocol-level integrations.

- **Audit Logger**: Tracks all signature requests and failures.
- **Key Manager**: Handles secure AES-GCM keystore loading and unloading.
- **Policy Engine**: Evaluates all requested transactions against configured spending limits.
- **Transaction Builder**: Standardizes Solana transaction packing and signing logic.
- **x402 Client/Server**: Protocols for pay-per-request HTTP headers.
